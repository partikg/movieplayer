"# movieplayer" 

Links:

Site (Vercel): movieplayer-site2.vercel.app

Backend API (Render): https://movieplayer-backend.onrender.com

Admin Panel (Vercel): movieplayer-adminpanel.vercel.app

------------------------------------------------------------

Project Structure:

movieplayer/

  api/              - Backend (Express + MongoDB)

  adminpanel/       - Admin Panel (React / Vite)
  
  site/             - User Site (Next.js)



------------------------------------------------------------

Tech Stack

Frontend (Site): Next.js, React

Admin Panel: React (Vite)

Backend: Node.js, Express

Database: MongoDB Atlas

Deployment:

Backend → Render

Admin Panel → Vercel

Site → Vercel



---------------------------------------------------------------

Run Project Locally

Backend (API): nodemon server.js

Admin Panel and site : npm run dev
